# 누리코리아 알림 플러그인

누리코리아(nurikorea.com)에서 발송하는 공식 공지·업데이트·보안 알림을 누리보드 관리자 화면에 자동으로 표시합니다.

## 기능
- 관리자 페이지 상단 배너 (최고 긴급도 알림)
- 우측 상단 종모양 아이콘 + 미읽음 카운트
- 드롭다운으로 전체 알림 목록 보기
- 개별 "읽음" 처리
- 6시간 간격 자동 체크 (서버 부담 없음)

## 설치 방법
1. 이 폴더 전체를 ZIP으로 압축
2. 누리보드 관리자 → 플러그인 → ZIP 업로드
3. 활성화

## 파일 구성
- `plugin.json` — 플러그인 메타데이터
- `plugin.php` — 메인 로직 (API 호출, 배너 렌더링)
- `mark-read.php` — 읽음 처리 엔드포인트
- `cache.json` — 자동 생성되는 로컬 캐시
- `read.json` — 읽음 처리된 알림 ID 목록

## API 정책
- 엔드포인트: `https://nurikorea.com/api/announcements.php`
- 메소드: GET
- 캐시: 6시간 (서버 부담 최소화)
- 응답: JSON (UTF-8)

## 보안
- HTTPS 필수
- CORS 허용
- 읽기 전용 API (누리코리아 서버에서만 알림 생성 가능)
